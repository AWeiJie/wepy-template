
## Build Setup

``` bash
# install dependencies
npm install # or yarn

# serve in dev mode, with hot reload at devtools
npm run dev

# build for production
npm run build
```

## 源码目录介绍
```
./src
├── pages                                  // 页面文件
├── components                             // 页面组件
├── common                                 // 公共文件
│   ├── api.js                             // 全局所有请求接口
│   ├── request.js                         // request请求封装 需替换请求的接口域名
│   ├── utils.js                           // 全局共用方法
└── images                                 // 图片文件

```
